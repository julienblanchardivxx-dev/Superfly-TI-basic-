# Superfly-TI-basic-
I remade Superfly in TI basic. A home screen game, in pure basic for
16x8 and 26x10 caricature TI calculators.

There is a main program called Superfly and 2 sub programs named Z0 and Z1.

Thank you so very much for trying my game. I've spent months learning and
making my code better and I'm very happy to release my star child of a project.
Thank you Nathan for the inspiration and thank you Lila for the intense testing
and getting the high score of 271 at round 12.
